# 0x01 — www

Landing page for [0x01.world](https://0x01.world).

Plain HTML/CSS. No framework, no build step.

Hosted on GitHub Pages → `zerox1.github.io` with custom domain `0x01.world`.

## Local preview

```bash
open index.html
```

## Deploy

Push to `main` → GitHub Pages serves automatically.

Custom domain: add `0x01.world` in repo Settings → Pages → Custom domain.
Create a `CNAME` file if GitHub doesn't do it automatically:

```
0x01.world
```
